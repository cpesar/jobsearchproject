In Demand Skills

I have compiled a list of in demand job skills which I have researched in recent job postings. The jobs I researched included: Web developer, Front-end developer, Back-end developer, Javascript developer, and Node developer

Common Skills in Job Postings:




Hyper Text Markup Language (HTML), is the common markup language for developing a web page. HTML elements are the basic building blocks of HTML pages. HTML elements are identified by tags and include opening and closing brackets. Some common HTML tags include: html (opening and closing), head (opening and closing), body (opening and closing), header (h1, openeing and closing), image (img, opening and closing) It should be noted that there is always an opening and closing tag. HTML can be further enhanced using scripting languauges such as CSS (cascading style sheets) and Javascript.




Simply put, Javascript is a scripting language for the web used to create and control dynamic web content. Javascript gives web pages interactive elements that engage the user by converting it from a static one to an interactive one. Javascript gives the user the ability to dynamically interact with a webpage. For example, Javascript gives the user the ability to enter their email address and add animated graphics, such as clickable links. Javascript is the scripting languauge that allows a user to refresh a webpage. Some other exciting things you can do with Javascript include: building web servers, mobile & web apps, smart watches, and autonomous flying drones.




Maintained by Facebook and a community of individual developers, React.js is is frontend, open source Javascript library. React.js gives web developers the ability to build interactive websites. React.js is a super useful Javascript library that specializes in helping web developers build interactive user interfaces. User interfaces include on screen menus, search bars, buttons or anything else that someone uses to interact with a website or app. React is exclusively frontend/client facing.




Typscript is a programming language developed by Microsoft. Used in the developemt of largescale applications, Typscript transcompiles to Javascript. The main goal of Typescript is to make Javascript more efficient by catching mistakes in code early on. Typescript can be used to develop applications for both client side (frontend) and server side (backend) programs. Some drawbacks to Typescript include the time it takes to debug which increases development time, as well as an increase in time it takes to bring on new developers.




Angular is an open source Javascript based client facing framework developed and maintained by Google and a community of individuals and corporations.




MongDB is an open source database management program, primarily used for the management of large data sets. MongDB's architecture is made of documents and collections, as opposed to tables and rows seen in relational databases.




Node.js is a a Javascript runtime enviroment that includes everything you need to execute a program written in Javascript. It allows Javascript to be run as a standalone option as well as in your browser. According to the Node.ja website, "Node.js® is a JavaScript runtime built on Chrome’s V8 JavaScript engine. Node.js uses an event-driven, non-blocking I/O model that makes it lightweight and efficient. Node.js’ package ecosystem, npm, is the largest ecosystem of open source libraries in the world."




PHP originally stood for 'Personal Home Pages', however it has since been known as Hypertext pre processor. PHP is a backend scripting language. Backend scripting languages are interperted on the server side. By comparison, Javascript is a frontend (client side) scripting language. It is reomended that you learn some HTML basics before diving into PHP. Some advantages of PHP are: that it is open source and free, is regularly updated and is easy to learn.




AWS (Amazon Web Services), is a subsidiary of Amazon that provides cloud computing platforms to individuals, companies, and government entities. The services that AWS provides include: cloud computing, storage, networking, database analytics, application services, mobile developer tools and much, much more.




SQL(Structured Query Language), is a programming languauge that is designed to retrieve information from databases. Oracle, Sybase, Microsoft SQL Server are some common relational database management systems that use SQL. Some jobs that use SQL's the most include: Business analyst, Software engineer, Systems administrator, and Quality assurance testers.




Designed by Microsoft C# (pronounced C sharp), is a versatile programming language. It is used to create programs and applications such as: mobile applications, desktop applications, cloud based services, websites, enterprise software and games. However, C# is the most common in game development. The greatest advantage that C# boasts is it's ability to save development time by dramatically reducing the time it takes to debug. C# draws some comparisons to Java in that they both are object oriented and use similar 'curly brace' syntax.
